Category: Analytics

## What it does

Scores any X draft 0–100 across six dimensions (hook, length, question, emotion, format, specificity) and suggests specific rewrites. Live recompute as you type.

## Why it exists

Most drafts fail for one of six predictable reasons. This tool names them before you hit post, so you can fix in 30 seconds instead of finding out from zero engagement.

## Tech stack

- Vanilla JS + inline SVG arc
- No dependencies, no build step

## Install

```bash
cd tools/04-pre-post-virality-scorer
open index.html
```

## Usage

1. Paste or type your draft
2. Watch the score arc fill in real time
3. Apply the suggestions — most add 10–20 points each
