# Security policy

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a vulnerability

Please report suspected vulnerabilities privately by emailing
**security@agentmindcloud.com**. Do not open a public GitHub issue for
security reports.

Include in your report:

- A description of the issue and the impact you observed.
- Steps to reproduce, ideally with a minimal proof of concept.
- The xlOS version, OS, and Python version where you saw it.

## Response commitment

We aim to acknowledge new reports within **5 business days**, confirm the
issue or request more information, and coordinate a fix and disclosure
timeline with the reporter.
