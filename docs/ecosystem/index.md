# Ecosystem

xlOS is a runtime around the open `grok-install` spec. It plays well
with the broader AI-agent ecosystem.

- **[xAI SDK](xai-sdk.md)** — the default runtime. xlOS wraps the SDK
  rather than re-implementing it.
- **[LiteLLM](litellm.md)** — use any provider (OpenAI, Anthropic,
  Ollama) behind a unified interface.
- **[Semantic Kernel](semantic-kernel.md)** — bridge xlOS agents into
  the Microsoft Semantic Kernel planner.
